Rasterize.js

Simple 2d rasterization algorithms implemented in JavaScript

Playground


Usage

Rasterized line from ( x0, y0 ) to ( x1, y1 ): var points = rasterize.lineB( x0, y0, x1, y1 );

Rasterized circle with origin in point ( x0, y0 ) and radius r: var points = rasterize.circleB( x0, y0, r );
